# 🎥 Video Call Web Application

A full-stack **real-time video calling web application** built using **Spring Boot (Backend)** and **HTML, CSS, JavaScript (Frontend)** with **ZEGOCLOUD integration** for video conferencing.

---

## ✨ Features

- 🔐 User Registration & Login
- 👥 View Connected Users (Online/Offline)
- 📹 Create & Join Video Meetings
- 🔗 Share meeting links
- 🔄 Real-time UI updates
- 🎨 Modern UI (Dark Theme)

---

## 🛠️ Tech Stack

### 💻 Frontend
- HTML5
- CSS3 (Custom UI)
- JavaScript (Vanilla JS)

### ⚙️ Backend
- Java Spring Boot
- REST APIs
- Maven

### ☁️ Video Integration
- ZEGOCLOUD UI Kit
